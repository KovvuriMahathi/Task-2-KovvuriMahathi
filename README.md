# Week 2 - Backend API Development

## Description
This project demonstrates a simple backend API using Node.js and Express.js.

## Features
- GET /students
- POST /students
- Input validation
- JSON responses

## Technologies Used
- Node.js
- Express.js
- Postman

## How to Run

1. Install dependencies

```bash
npm install
```

2. Start the server

```bash
node server.js
```

3. Open Postman and test:

GET http://localhost:3000/students

POST http://localhost:3000/students
